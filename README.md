# Сложно сосредоточиться

Адаптивный сайт с поддержкой светлой и тёмной тем оформления, а также автоматическим переключением в зависимости от системных настроек пользователя.

## Технологии

- HTML5
- CSS (Grid, Flexbox, Custom Properties, Media Queries)
- JavaScript

## Функциональность

- Адаптивная вёрстка (mobile-first: 375px → 768px → 1024px)
- Тёмная и светлая цветовые схемы
- Автоматическое определение темы через `prefers-color-scheme`
- Принудительное переключение темы кнопками
- Фиксированное фоновое изображение
- Ленивая загрузка изображений
